La etiqueta video: Para especificar la ubicacion del archivo que se reproducidira, puede usar el elemnto <source> dentro del elemento <video>. (Esto debe reemplazar el atributo src en la etiqueta <video< de apertura.) Tambien puede utilizar varios elemntos <source> para especificar que el video esta disponible en diferentes formatos. (Debido a un error en el ipad, debe proporcionar el video MP4 como el primer formato. De lo contrario, puede que no inicie.)

src: este atributo especifica la ruta de acceso al video.

type se debe utilizar este atributo para decirle al navegador que formato es el video. De lo contrario, descargara parte del video para ver si puede reproducir el archivo (que tomara tiempo y ancho de banda).

codecs: el codec que se uso para codificar el video se sumistra dentro del atributo type.Tenga en cuenta el uso de comillas simples, asi como comillas dobles en el atributo type, Cuando se suministra
