Conceptos Clave en el posicionamiento de Elemento.

Construccion de Bloques

CSS trata cada elemento HTML como si estuviera en su propio cuadro. Este cuadro sera un block-level o un inline-box fluyen entre el texto circundante. puede controlar la cantidad de espacio que ocupa cada caja estableciendo el ancho de los cuadros (y aveces la altura tambien). para separar los cuadros, puede usar bordes, margenes, relleno y colores sde fondo.

Normal flow

Cada elemento de block-level se encuentra en la parte superior de la siguiente. puesto que esta en la forma predeterminadaen que los exploradores tratan los elementos HTML, no se necesita una propiedad CSS para indicar que los elemntos deben aparecer en el flujo normal.
