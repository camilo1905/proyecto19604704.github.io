CSS selectors

Hay muchos tipos diferentes de selector CSS que le permiten orientar las reglas a elementos especificos en un documento HTML.

Los selectores CSS distinguen entre maYusculas y minisculas, por lo que deben coincidir exactamente con los nombres de elemento y los valores de atributo.

Tipos de selectores
Selector Significado Ejemplo
Universal Selector Se aplica a todos los elementos del documento * {} Se dirige a todos los elementos de la pagina
Type selector Coinciden con los nombres de elemento h1, h2, h3 {}corresponde a los elementos <h1>, <h2>, <h3>
Class Selector coinciden con un elemento cuyo atributo de clase tiene un valor que corresponda con el especificado despues del simbolo de punto "." . nota {} se dirige a cualquier elemento cuyo atributo de clases tenga un valor de nota
Id Selector Coinciden con un elemento cuyo ID atributo tiene un valor que corresponda con el especificado despues del simbolo de hash "#" #introduction {} se dirige al elemento cuyo atributo ID tiene un valor de introduction
