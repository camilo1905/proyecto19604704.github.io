background-color-color de fondo

Si no es especifica un color de fondo, el fondo es transparente

CSS trata cada elemnto HTML como un cuadro y la propiedad background-color establece el color del fondo para el cuadro. se puede especificar un color de fondo de las mismas tres formas en que pueda espedificar colores de primer plano:valores RGB, codigo Hexadecimal y nombres de colores. Tambien hemos utilizado la propiedad padding para separar el texto de los bordes de los cuadros. Esto hace que sea mas facil de leer.
