cada elemento HTML puede llevar el atributo id. se utliza para identificar de forma unica ese elemento de otros elementos de la pagina. su valor debe comenzar con una letra o un guion bajo (no un numero o cualquier otro caracter). Es importante que no hay dos elementos en la misma pagina tienen el mismo valor para sus atributos ID (de lo contrario, el valor, el valor ya no es unico)

dar un elemento una identidad unica le permite darle un estilo diferente que cualquier otra instancia del mismo elemento en la pagina

por ejemplo es posible que desee asignar un parrafo dentro de la pagina (quizas un parrafo que contiene una cotizacion de extraccion) un estilo diferente que todods los demas parrafos. en el ejempplo, el parrafo con el atributo ID cuyo valor es pullquote se hace en mayuscula usando CSS.
